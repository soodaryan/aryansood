## Getting Started

To run this website locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/soodaryan/aryansood.git

2. Navigate into the project folder:

   ```bash
   cd aryansood
   ```
3. Open `index.html` in your favorite browser.
